# Images

Add your build photos here. README.md references these filenames:

| Filename          | Description                                      |
|-------------------|--------------------------------------------------|
| `assembled.jpg`   | v2 fully assembled — front view with UI on TFT   |
| `internals.jpg`   | Inside — ESP32 perf board + TP4056 + 18650 cell  |
| `cad_body.png`    | Onshape screenshot — main enclosure body         |
| `cad_knob.png`    | Onshape screenshot — encoder knob                |

Rename and drop your photos here, then:

```bash
git add images/
git commit -m "Add v2 build photos"
git push
```
